## 3. Problem → Solution 구조

**섹션 제목**  
왜 지금 자람이인가

**한 줄 요약**  
데이터 단절, 골든타임 손실, 비용 대비 불확실한 효과 — 세 가지 구조적 문제를 해결합니다.

**1) 데이터 단절**  
가정·학교·센터·병원 기록이 분절되어:  
- 기관별 기록 분절  
- 장기 패턴 추적 불가  
- 중복 검사 발생  
- 협업 어려움  

**2) 골든타임 손실**  
발달장애는 **3~7세 조기개입**이 핵심이지만:  
- 정보 비대칭  
- 기관 간 맥락 공유 부족  
→ 상당수 가정이 개입 시기를 놓침  

**3) 비용 대비 효과 불확실**  
- 월 200~300만원 치료비  
- 맞춤 계획 부재  
- 효과 검증 어려움  

**Solution 한 줄 (섹션 마무리)**  
자람이는 표준 데이터 인프라와 AI 분석으로 위 세 가지를 구조적으로 해결합니다.

## 4. How It Works (시스템 작동 구조)

**섹션 제목**  
AI 디지털케어로그 작동 구조

**한 줄 요약**  
입력 → 표준화 → AI 분석 → 실시간 환류. 데이터가 쌓일수록 정확도가 높아지는 강화학습 구조입니다.

**4단계 파이프라인**

1. **데이터 수집**  
   - 가정: 부모 관찰 기록  
   - 학교: 특수교사 IEP  
   - 센터: 치료사 활동 기록  
   - 병원: 진단·약물 기록  

2. **통합·표준화**  
   - 표준 스키마로 통합  
   - 시계열 정렬  
   - 시간·장소·맥락 자동 태깅  
   - 중복 제거·누락 보완  

3. **AI 분석**  
   - 패턴 인식·예측  
   - 발달 추이 시각화  
   - 이상 징후 조기 탐지  
   - 개인별 최적 전략 추천  

4. **실시간 환류**  
   - 부모: 가정 실천 과제  
   - 치료사: 전략 조정 제안  
   - 교사: 학교 환경 팁  
   - 의사: 진료·처방 참고  

**강조 문장**  
자람이는 단순 기록 시스템이 아니라, **데이터가 쌓일수록 정확도가 높아지는 강화학습 구조**입니다.
