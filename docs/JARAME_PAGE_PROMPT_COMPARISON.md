### 6. 문제 정의 — 불릿 상세
- **데이터 단절**: 중복 검사 발생, 모순된 처방, 장기 패턴 추적 불가, 치료 효과 객관 평가 어려움
- **골든타임**: 치료사 부족, 정보 비대칭, 기관 간 맥락 공유 부족 → 상당수 가정이 개입 시기를 놓침
- **비용**: 월 200~300만원, 맞춤 계획 부재, 효과 검증 어려움
