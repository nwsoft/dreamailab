## 5단계 플랫폼 (`platformSteps`)

1. **학습·행동 로그 수집** — 과제·진단·반응 시간·이탈 지점
2. **분석** — 성취·오개념·주의 패턴 (구간 단위)
3. **개인화 경로 생성** — 복습·신규·보정 비율
4. **실행** — 퀴즈·루틴·리포트 (교사·부모 확인 후 적용)
5. **환류(재학습)** — 실행 결과 → 로그 → 경로 수정
